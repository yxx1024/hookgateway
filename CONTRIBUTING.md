# 贡献指南

感谢你愿意贡献 HookGateway。请在提交前阅读以下约定。

## 开发环境
- JDK 17
- Maven 3.x
- （可选）Docker / Redis / MySQL

## 开发流程
1) Fork 仓库并创建分支（如 `feature/xxx` 或 `fix/xxx`）。  
2) 小步提交，PR 描述清楚背景与动机。  
3) 运行测试：`mvn test`。  
4) 必要时更新文档与配置说明。  

## 代码要求
- 遵循现有风格，避免引入不必要的复杂性。  
- 新增功能请尽量补充测试。  
- 安全相关变更需说明威胁模型与边界。  

## 提交检查清单
- [ ] 代码可读、逻辑清晰  
- [ ] 测试通过或说明原因  
- [ ] 文档与配置同步更新  

## 行为准则
参与贡献即表示遵守 [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)。
